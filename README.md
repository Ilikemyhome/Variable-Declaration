# Understanding `const` and `let` in JavaScript

This project demonstrates how to correctly replace `var` with `let` or `const` in modern JavaScript. Each example includes comments explaining why a specific keyword is appropriate and what issues would occur if the other keyword were used.

## Overview
The goal of this assignment is to understand:
- When to use `let` vs `const`
- How variable reassignment affects keyword choice
- Why `var` should be avoided in modern JavaScript
- How block scoping improves code safety

## Concepts Covered
- Variable declaration keywords (`var`, `let`, `const`)
- Reassignment vs mutation
- Block scope vs function scope
- Best practices for modern JavaScript
